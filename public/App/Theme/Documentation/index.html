Router

Template uses very simple routing system. Router checks for pages in 'pages' directory. If the proper include file is found, it will be displayed. If its not found, router will return 404 status code and display appropriate page.

Examples

Route: /faq - Router opens '/pages/' directory and searches for file or faq.php

Route: /properties - Router opens '/pages/' directory and searches for file or directory named properties.php. If available, page is displayed. If nothing is found, router will make final try an search for index.php in context of /pages/properties/ folder. If index.php is found, its displayed.

Route: /properties/property-detail - Router opens '/pages/properties/' directory and searches for file named property-detail.php. If available, page is displayed.

Route: /properties/i-dont-exist - Router opens '/pages/properties/' directory and searches for file named i-dont-exist.php. File will not be found and router will return 404 status code.

Templates

Template is divided into 4 main regions - Header, Footer, Main, Sidebar.
Appropriate directories can be found if 'templates' directory.
Each region contains set of reusable blocks - pieces of content, forms, listings.. etc.

  • Header
    • block-map-extended-search.php - contains code for extended map vertical filter
    • block-map-filter-horizontal.php - contains code for extended map horizontal filter
    • navigation.php - contains code for displaying main navigation of theme.
    • top.php - contains code for displaying language switcher, logo and social menu links.
  • Main
    • block-carousel-properties.php - displays properties in carousel form - displayed on frontpage
    • block-featured-properties.php - displays grid of "featured" properties - displayed on frontpage
    • block-features.php - displays grid of site features - displayed on frontpage
    • block-partner.php - displays grid of site partners - displayed on frontpage
    • block-recent-properties.php - displays grid of "recent" properties - displayed on frontpage
  • Sidebar
    • block-agencies.php - displays list of agencies for sidebar
    • block-enquire-now.php - displays "enquire now" form which is displayed on property detail page
    • block-extended-search.php - displays extended search form
    • block-most-recent.php - displays list of "most recent" properties
    • block-quick-search.php - displays simple search form - one field
  • Footer - footer looks the same in the whole theme, thus it was not divided into smaller pieces.

Assets

All the required assets are included in main index.php file - css, js.

The main css files is located in '/assets/css/properta.css'. However this is generated from sass files located in '/assets/scss'.

If you dont have experience with sass and compass you can simply replace your properta.css (This file generated and thus its in minified version) with properta.ext.css - which is nicely structured.
